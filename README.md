# Rahmani Malabre — Personal Portfolio

Personal portfolio of Rahmani (Azariah) Malabre — Applied AI & Data Science.

Live: **rahmanimalabre.com** *(once domain is connected)*

## Stack
Static HTML / CSS / JS. No build step.
- `index.html` — hero with tilted 3D planetary orbit (Claude / OpenAI / Gemini / Perplexity logos), custom cursor, meteor easter egg
- `CV.html` — full curriculum vitae sub-page
- `resume.pdf` — downloadable resume
- React 18 via UMD + Babel standalone for the JSX components (`app.jsx`, `glyphs.jsx`, `tweaks-panel.jsx`)

## Deploying
Zero config — Vercel auto-detects a static site. Just import the repo.

## SEO
- `<title>`, meta description, Open Graph, Twitter card, JSON-LD `Person` schema
- `favicon.svg`, `og.png`
- `sitemap.xml`, `robots.txt`

After deploying, submit to **Google Search Console** at https://search.google.com/search-console to get indexed.
